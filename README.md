# 🧮 Java Calculator CLI

Uma aplicação de calculadora baseada em console, desenvolvida para exercitar conceitos fundamentais de **Programação Orientada a Objetos (POO)** e manipulação de fluxos em Java.

## 🚀 Funcionalidades

* **Operações Básicas:** Soma, Subtração, Multiplicação e Divisão.
* **Saudação Inteligente:** O sistema identifica o horário local e cumprimenta o usuário com "Bom dia", "Boa tarde" ou "Boa noite".
* **Tratamento de Exceções:** 
    * Proteção contra divisão por zero.
    * Validação de caracteres de operação inválidos.
* **Fluxo Contínuo:** Permite realizar múltiplos cálculos ou encerrar o programa digitando "F".

## 🛠️ Tecnologias Utilizadas

* **Java 17+**
* **Scanner API** (Interação via terminal)
* **Java Time API** (Saudação dinâmica)

## 📦 Como rodar o projeto

1. Certifique-se de ter o **JDK** instalado.
2. Clone o repositório:
```bash
git clone [https://github.com/pedrohbhrj/Calculadora.git]
```
3. Compile os arquivos:
```bash
javac br/com/calculadora/Calculadora.java br/com/Main.java
```
4. Execute a aplicação:
```bash
java br.com.Main
```
